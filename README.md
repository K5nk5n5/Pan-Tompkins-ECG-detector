# Pan-Tompkins-ECG-detector
A MATLAB implementation of the Pan-Tompkins algorithm for detecting R-peaks in 12-lead ECG signals.
Computes RR intervals, heart rate, and basic HRV metrics (SDNN, RMSSD, pNN50).

# Instructions:
- place '.mat' file in the same folder as 'pan_tompkins.m'
- in the script, edit the file name and row number in lines 7,8
- in the cirrent code, Fs=1000Hz, edit if any other Fs is being used
- Signal Processing Toolbox is required
